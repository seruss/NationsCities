Jak grać

Przewodnik po grze

Witaj w mobilnej wersji Państwa, Miasta! Graj ze znajomymi w czasie rzeczywistym, unikaj oszustw i baw się dobrze.

Przebieg rundy

casino

1. Losowanie litery

System losuje literę dla wszystkich graczy.

timer

2. Walka z czasem

Wpisz odpowiedzi zanim upłynie czas rundy.

gavel

3. Weryfikacja

System Anty-Cheat i gracze sprawdzają słowa.

Szczegóły rozgrywki

rule

Podstawowe zasady

expand_more

Gra składa się z określonej liczby rund. W każdej rundzie liczy się refleks. Kto pierwszy wciśnie "Stop", zatrzymuje czas dla innych (jeśli ta opcja jest włączona) lub gra toczy się do końca odliczania zegara.

category

Kategorie

expand_more

Standardowe kategorie to: Państwo, Miasto, Roślina, Zwierzę, Imię, Rzecz.

Gospodarz pokoju może dodawać własne, niestandardowe kategorie (np. Marka Samochodu, Tytuł Filmu).

security

System Anty-Cheat

expand_more

Nasza gra wykorzystuje inteligentną weryfikację słów. Jeśli spróbujesz wpisać losowe znaki lub słowa nieistniejące, system automatycznie je odrzuci i nie przyzna punktów. Dodatkowo inni gracze mogą zgłaszać wątpliwe odpowiedzi w fazie głosowania.

scoreboard

Punktacja

expand_more
  • Unikalna odpowiedź 15 pkt
  • Poprawna odpowiedź 10 pkt
  • Powtórzona odpowiedź 5 pkt
  • Brak lub błędna 0 pkt
